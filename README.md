## Notes

### Keyboard Controls
| Key        | Action                                                                 |
|------------|------------------------------------------------------------------------|
| `.`        | Select **next** component in the hierarchy                             |
| `,`        | Select **previous** component in the hierarchy                         |
| `x`        | Rotate the selected component **forward around X-axis**                |
| `X`        | Rotate the selected component **backward around X-axis**               |
| `y`        | Rotate the selected component **forward around Y-axis**                |
| `Y`        | Rotate the selected component **backward around Y-axis**               |
| `z`        | Rotate the selected component **forward around Z-axis**                |
| `Z`        | Rotate the selected component **backward around Z-axis**               |

### Mouse Controls

| Action                                   | Effect                                                                |
|-----------------------------------------|------------------------------------------------------------------------|
| **Left-click + drag horizontally**       | Rotate the **root component (`Torso`) left or right**                 |
| **Left-click + drag vertically**         | Rotate the **root component (`Torso`) up or down**                    |
